Asset Analysis API

Consumer Credit Scoring

Overview

Analyzes customer assets and investments to assess financial strength.

Endpoint

POST https://mycfo.club/stori/api/customer/asset/analyze-json/

Authentication

X-API-Key: your_api_key_here
Content-Type: application/json

Request Body

Sample Input JSON

{
  "customer_name": "Rahul Sharma",
  "customer_pan": "ABCDE1234F",
  "asset_data": {
    "real_estate": [
      {
        "property_type": "residential",
        "market_value": 8000000,
        "location": "Mumbai",
        "ownership": "owned"
      }
    ],
    "vehicles": [
      {
        "vehicle_type": "car",
        "market_value": 800000,
        "ownership": "owned"
      }
    ],
    "investments": {
      "mutual_funds": 500000,
      "stocks": 300000,
      "fixed_deposits": 1000000,
      "ppf": 400000,
      "gold": 200000
    }
  }
}

Response

Sample Output JSON

{
  "success": true,
  "data": {
    "features": {
      "total_assets": 11200000,
      "liquid_assets": 2400000,
      "real_estate_value": 8000000,
      "investment_portfolio": 2400000,
      "asset_diversity_score": 0.85
    }
  }
}

Usage Notes

  • Asset analysis is optional but adds strength to credit profile
  • Store the returned features JSON for final credit scoring
